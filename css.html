:not()

This pseudo-class represents elements that do not match a list of selectors. Since it prevents specific items from being selected, it is known as the negation pseudo-class.

Example:

p:not(.irrelevant) {
    font-weight: bold;
}
^

@media

The @media CSS at-rule can be used to apply part of a style sheet based on the result of one or more media queries. With it, you specify a media query and a block of CSS to apply to the document if and only if the media query matches the device on which the content is being used.

Example:

@media screen and (min-width: 900px) {
    article {
        padding: 1rem 3rem;
    }
}
^

@supports

This CSS at-rule lets you specify CSS declarations that depend on a browser's support for CSS features. Using this at-rule is commonly called a feature query. The rule must be placed at the top level of your code or nested inside any other conditional group at-rule.

Example:

@supports (display: flex) {
    .flex-container > * {
        text-shadow: 0 0 2px blue;
        float: none;
    }

    .flex-container {
        display: flex;
    }
}
^

aspect-ratio

This property makes it much easier to maintain consistent proportions for elements across different screen sizes. It eliminates the need for complicated CSS tricks or JavaScript solutions.

Example:

.responsive-element {
    width: 100%;
    aspect-ratio: 16/9;
}
^

box-shadow

Allows you to create shadows around elements, adding depth and dimension to your designs. You can customize the shadowâ€™s color, offset, blur radius, and spread.

Example:

.element {
    box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.2);
}
^

box-sizing and border-box

By default, the content-box setting adds padding and borders to the content width, which can complicate responsive designs. Using box-sizing with border-box ensures that the total width calculation is predictable, making layout adjustments much smoother.

Example:

/* Global box sizing reset */
* {
    box-sizing: border-box;
}

/* Use content-box only for specific cases */
.special-element {
    box-sizing: content-box;
}
^

clamp()

The clamp() CSS function clamps a middle value within a range of values between a defined minimum bound and a maximum bound. The function takes three parameters: a minimum value, a preferred value, and a maximum allowed value.

Example:

.demo {
    font-size: clamp(1rem, 2.5vw, 2rem);
}
^

clip-path

The clip-path property allows you to shape elements using paths, creating custom shapes and masks. You can use various path syntaxes, such as circles, ellipses, polygons, and more.

Example:

.element {
    clip-path: circle(50% at 50% 50%);
}
^

column-count and column-gap

These properties can be used to create multi-column layouts, similar to newspapers or magazines. You can control the number of columns and the spacing between them.

Example:

.content {
    column-count: 3;
    column-gap: 20px;
}
^

counter-reset and counter-increment

These properties can be used to create custom counters for numbering elements within a document. For example, you can use them to create numbered lists, page numbers, or other sequential elements.

Example:

body {
    counter-reset: my-sec-counter;
    padding: 50px;
}

h2::before {
    counter-increment: my-sec-counter;
    content: "Section " counter(my-sec-counter) ". ";
}
^

mix-blend-mode

The mix-blend-mode property allows you to blend elements together using different blending modes. This can create interesting visual effects, such as multiplying, screening, overlaying, or differencing elements.

Example:

.element1 {
    width: 100px;
    height: 100px;
    background-color: red;
}

.element2 {
    width: 50px;
    height: 50px;
    margin: auto;
    background-color: blue;
    mix-blend-mode: multiply;
}
^

overflow

The overflow CSS shorthand property sets the desired behavior when content does not fit in the element's padding box (overflows) in the horizontal and/or vertical direction.

Example:

div {
/* Keyword values */
overflow: visible;
overflow: hidden;
overflow: clip;
overflow: scroll;
overflow: auto;
overflow: hidden visible;

/* Global values */
overflow: inherit;
overflow: initial;
overflow: revert;
overflow: revert-layer;
overflow: unset;
}
^

perspective and transform

The perspective and transform properties can be used to create 3D effects, such as rotations, translations, and scaling. By combining these properties, you can achieve visually stunning effects.

Example:

.element {
    perspective: 500px;
    transform: rotateZ(45deg);
}
^

vh and vw

These properties are perfect for creating full-screen elements such as hero sections or overlay menus. Pairing percentages with viewport units enhances layout flexibility.

1vw is relative to 1% of the width of the viewport
1vh is relative to 1% of the height of the viewport

Example:

.sidebar {
    width: min(30%, 300px);
    padding: max(2vw, 1rem);
}
^

word-break

The word-break CSS property sets whether line breaks appear wherever the text would otherwise overflow its content box.

Example:

/* Keyword values */
word-break: normal;
word-break: break-all;
word-break: keep-all;
word-break: auto-phrase; /* experimental */
word-break: break-word; /* deprecated */

/* Global values */
word-break: inherit;
word-break: initial;
word-break: revert;
word-break: revert-layer;
word-break: unset;
^
