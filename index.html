<abbr>

Definition: Represents an abbreviation or acronym, with an optional title attribute for the full term.

Usage: Use this tag to provide additional information about abbreviations or acronyms.

Example:

<abbr title="Hypertext Markup Language">HTML</abbr>

Special Attributes:
title - Specifies the full term for the abbreviation.

^

<bdi>

Definition: Isolates a part of text that might be formatted in a different direction from other text outside it.

Usage: Use this tag to prevent text from being incorrectly formatted when mixed with text in a different direction.

Example:

<bdi>مرحبا</bdi> world
^

<bdo>

Definition: Overrides the current text direction.

Usage: Use this tag to change the text direction of its content.

Example:

<bdo dir="rtl">This text is right-to-left.</bdo>

Special Attributes:
dir - Specifies the text direction (ltr or rtl).

^

<cite>

Definition: Represents the title of a work (e.g., a book, a song, a movie).

Usage: Use this tag to cite the title of a work.

Example:

<cite>The Great Gatsby</cite>
^

<datalist>

Definition: Contains a set of <option> elements that represent the permissible or suggested options available to other controls.

Usage: Use this tag to provide a list of predefined options for an input element.

Example:

<input list="browsers">
    <datalist id="browsers">
        <option value="Chrome">
        <option value="Firefox">
        <option value="Safari">
        <option value="Edge">
    </datalist>
^

<details>

Definition: Creates a disclosure widget that users can open and close.

Usage: Use this tag to create a collapsible section of content.

Example:

<details>
    <summary>More info</summary>
    <p>This is the detailed content.</p>
</details>
^

<dialog>

Definition: Represents a dialog box or other interactive component, such as an inspector or window.

Usage: Use this tag to create a dialog box.

Example:

<dialog open>
    <p>This is an open dialog box.</p>
</dialog>

Special Attributes:
open - Specifies that the dialog is open.

^

<fieldset>

Definition: Groups related elements in a form and draws a box around them.

Usage: Use this tag to group related form elements.

Example:

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</fieldset>
^

<keygen>

Definition: Generates a key pair.

Usage: Use this tag to generate a key pair for forms.

Example:

<keygen name="name">

Special Attributes:
name - Specifies the name of the key pair.

^

<legend>

Definition: Represents a caption for the content of its parent <fieldset>.

Usage: Use this tag to provide a caption for a fieldset.

Example:

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</fieldset>
^

<mark>

Definition: Highlights text.

Usage: Use this tag to highlight text.

Example:

<mark>Highlighted text</mark>
^

<meter>

Definition: Represents a scalar measurement within a known range (e.g., disk usage).

Usage: Use this tag to display a measurement within a range.

Example:

<meter value="0.6">60%</meter>

Special Attributes:
value - Specifies the current value.
min - Specifies the minimum value.
max - Specifies the maximum value.
low - Specifies the low value range.
high - Specifies the high value range.
optimum - Specifies the optimum value.

^

<optgroup>

Definition: Groups options within a <select> element.

Usage: Use this tag to group related options in a dropdown list.

Example:

<select>
    <optgroup label="Fruits">
        <option value="apple">Apple</option>
        <option value="banana">Banana</option>
    </optgroup>
    <optgroup label="Vegetables">
        <option value="carrot">Carrot</option>
        <option value="broccoli">Broccoli</option>
    </optgroup>
</select>

Special Attributes:
label - Specifies the label for the option group.

^

<output>

Definition: Represents the result of a calculation or user action.

Usage: Use this tag to display the result of a calculation or user action.

Example:

<form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
    <input type="range" id="a" value="50"> +
    <input type="number" id="b" value="50">
    <output name="result" for="a b">100</output>
</form>

Special Attributes:
for - Specifies the IDs of the elements whose values are used in the calculation.

^

<picture>

Definition: Contains zero or more <source> elements and one <img> element to provide multiple versions of an image for different display/device scenarios.

Usage: Use this tag to provide different image sources for different devices or screen sizes.

Example:

<picture>
    <source media="(min-width: 800px)" srcset="large.jpg">
    <source media="(min-width: 400px)" srcset="medium.jpg">
    <img src="small.jpg" alt="Example image">
</picture>
^

<progress>

Definition: Displays the progress of a task.

Usage: Use this tag to show the progress of a task.

Example:

<progress value="70" max="100">70%</progress>

Special Attributes:
value - Specifies the current progress.
max - Specifies the maximum value of the progress.

^

<summary>

Definition: Defines a visible heading for the <details> element.

Usage: Use this tag to provide a summary or heading for a <details> element.

Example:

<details>
    <summary>More info</summary>
    <p>This is the detailed content.</p>
</details>
^

<template>

Definition: Holds client-side content that won’t be rendered when the page loads but can be instantiated later using JavaScript.

Usage: Use this tag to define HTML fragments that can be used later.

Example:

<template id="my-template">
    <div>This is a template</div>
</template>
<script>
    var template = document.getElementById('my-template');
    document.body.appendChild(template.content.cloneNode(true));
</script>
^

<time>

Definition: Represents a specific period in time.

Usage: Use this tag to mark up dates and times.

Example:

<time datetime="2024-12-17T10:56:12">December 17, 2024</time>

Special Attributes:
datetime - Specifies the date and time in a machine-readable format.

^

Usage: Use this tag to suggest a line break opportunity within text.

Example:
